ArkTS与WebView交互示例

本示例展示了如何在鸿蒙应用中使用ArkTS与WebView通过JS接口进行通信。ArkTS封装了可扩展的JS接口，H5页面可以通过这些接口与原生应用交互。

ArkTS页面实现

以下是完整的ArkTS页面代码，展示了如何封装可扩展的JS接口：

// WebViewPage.ets import webview from '@ohos.web.webview'; import prompt from '@ohos.prompt'; import router from '@ohos.router'; @Entry @Component struct WebViewPage { controller: webview.WebviewController = new webview.WebviewController(); jsBridge: JSBridge = new JSBridge(); aboutToAppear() { // 注册默认接口 this.jsBridge.registerHandler('showToast', this.showToast.bind(this)); this.jsBridge.registerHandler('showDialog', this.showDialog.bind(this)); this.jsBridge.registerHandler('getDeviceInfo', this.getDeviceInfo.bind(this)); this.jsBridge.registerHandler('goBack', this.goBack.bind(this)); // 添加JSBridge到WebView this.controller.addJavaScriptProxy({ object: this.jsBridge, name: 'JSBridge', methodList: ['onMessage', 'callHandler'] }); } // 显示Toast showToast(message: string) { prompt.showToast({ message: message, duration: 2000 }); return `Toast显示: ${message}`; } // 显示对话框 showDialog(title: string, message: string) { prompt.showDialog({ title: title, message: message, buttons: [{ text: '确定' }, { text: '取消' }] }); return `对话框显示: ${title} - ${message}`; } // 获取设备信息 getDeviceInfo() { const deviceInfo = { model: device.deviceModel, osVersion: device.osFullName, screen: `${window.getWindowWidth()}x${window.getWindowHeight()}`, language: i18n.getSystemLanguage() }; return JSON.stringify(deviceInfo); } // 返回上一页 goBack() { router.back(); return '返回操作执行'; } // 调用H5方法 callH5Method() { this.controller.runJavaScript("receiveFromNative('Hello from ArkTS!')"); } build() { Column() { // WebView组件 Web({ src: $rawfile('index.html'), controller: this.controller }) .onPageEnd(e => { console.log('页面加载完成'); }) .javaScriptAccess(true) .fileAccess(true) .domStorageAccess(true) .width('100%') .height('100%') } .width('100%') .height('100%') .padding(12) } } // 可扩展的JSBridge类 class JSBridge { private handlers: Map<string, Function> = new Map(); // 注册处理器 registerHandler(handlerName: string, handler: Function) { this.handlers.set(handlerName, handler); } // 调用处理器 callHandler(handlerName: string, ...args: any[]): any { const handler = this.handlers.get(handlerName); if (handler) { return handler(...args); } console.error(`未找到处理器: ${handlerName}`); return null; } // 通用消息处理器 onMessage(message: string) { console.log(`收到H5消息: ${message}`); try { const msg = JSON.parse(message); // 处理自定义消息 return '消息已接收'; } catch (e) { console.error('消息解析错误:', e); return '消息格式错误'; } } }

设计说明：

  • 可扩展架构 - 通过registerHandler方法可动态添加功能
  • 双向通信 - 支持ArkTS调用H5和H5调用ArkTS
  • 安全封装 - 使用类型安全的接口调用
  • 通用消息通道 - onMessage方法处理自定义消息

H5调试页面代码

以下是用于调试的H5页面代码，可直接在WebView中加载：

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>JSBridge调试页</title> <style> /* 样式代码与当前页面类似 */ </style> </head> <body> <div class="container"> <h1>JSBridge调试页面</h1> <div class="controls"> <button onclick="callJSBridge('showToast')">显示Toast</button> <button onclick="callJSBridge('showDialog')">显示对话框</button> <button onclick="callJSBridge('getDeviceInfo')">获取设备信息</button> <button onclick="callJSBridge('goBack')">返回上一页</button> </div> <div class="result-area"> <h3>调试结果：</h3> <div id="result"></div> </div> <script> // 调用JSBridge方法 function callJSBridge(method) { let result; switch(method) { case 'showToast': result = JSBridge.callHandler('showToast', 'Hello HarmonyOS!'); break; case 'showDialog': result = JSBridge.callHandler('showDialog', '提示', '操作成功'); break; case 'getDeviceInfo': result = JSBridge.callHandler('getDeviceInfo'); break; case 'goBack': result = JSBridge.callHandler('goBack'); break; } document.getElementById('result').innerText = result; } // 接收来自ArkTS的消息 function receiveFromNative(message) { const resultDiv = document.getElementById('result'); resultDiv.innerHTML = `<div class="log-entry success">收到ArkTS消息: ${message}</div>` + resultDiv.innerHTML; } // 初始化 window.addEventListener('load', () => { document.getElementById('result').innerHTML = '<div class="log-entry info">H5页面已加载完成，可以开始调试</div>'; }); </script> </div> </body> </html>
